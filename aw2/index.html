RESUMEN ETIQUETAS HTML-XHTML
Elementos de estructura
!DOCTYPE Especifica al navegador la codificacion de caracteres y la version HTML O XHTML que se esta utilizando
<"html">       <"/html"> Etiqueta que enmarca a todo el documento HTML
<"head">      <"/head"> Zona de cabecera de documento, etiqueta obligatoria
<"title">      <"/title"> Cuerpo del documento, etiqueta obligatoria
<"!-- comnetarios --"> Linea de comentarios

Elementos de estructura
<"hn">       <"/hn"> Cabeceras, n con valor de 1 a 6. 1 es el tamaño maximo de caracter
